# VideoOneLoadBot

Бесплатный Telegram-бот для скачивания видео с YouTube, TikTok, ВКонтакте и Instagram.

🌐 **Сайт:** https://vovkslo.github.io/

🤖 **Бот:** https://t.me/VideoOneLoadBot

## Возможности
- YouTube и YouTube Shorts — до 1080p
- TikTok — без водяного знака  
- ВКонтакте — публичные видео
- Instagram Reels

Без регистрации, без рекламы, до 2 ГБ.
